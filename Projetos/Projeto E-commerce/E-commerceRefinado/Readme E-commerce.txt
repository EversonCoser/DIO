Para resolver o primeiro item duas entidades foram criadas, uma para armazenar as
informações referentes a pessoa física e outra para a pessoa jurídica, essas entidades
estão se relacionando pela chave primária com o campo TipoCliente. A tabela pessoa 
física contém o cpf e pode conter outros atributos relacionados ao tipo Pessoa Física,
o mesmo acontece com a tabela pessoa juridica.

No item dois uma entidade chamada FormaPagamento foi criada para armazenar as
formas de pagamento do cliente, essa entidante se relaciona com a tabela pedido. Além disso,
uma entidade InformaçõesCartão foi criada para armazenar as informações referentes ao cartão do
cliente.

Para finalizar, no item três, temos a criação de uma entidade para representar a entrega,
ela se relaciona com a entidade Pedido.
